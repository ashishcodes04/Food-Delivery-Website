# 🍔 Food-Delivery-Website

A responsive Food Delivery Website built using HTML, CSS, and JavaScript. Users can browse food items, view menus, add items to the cart, and place orders through a clean and user-friendly interface.

# 🌐 Live Demo

🔗 **Food Delivery Website:**
https://github.com/ashishcodes04/Food-Delivery-Website.git

# 🚀 Features

   * Responsive design for mobile and desktop
   * Browse food categories
   * View food items with images and prices
   * Add and remove items from cart
   * Order summary page
   * Easy-to-use user interface
   * Fast and lightweight website
     
# 🛠️ Technologies Used

   * HTML5
   * CSS3
   * JavaScript
   * Git & GitHub
